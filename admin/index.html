Đăng nhập admin

Nhập mật khẩu quản trị để mở giao diện. Cùng mật khẩu này dùng được trên mọi thiết bị (kể cả điện thoại).

Mật khẩu được lưu dưới dạng SHA-256 hash bên trong file admin/index.html. Hash không đảo ngược được nên kể cả xem source code cũng không suy ra mật khẩu (với mật khẩu đủ mạnh).

Cài đặt lần đầu

File admin/index.html chưa được cấu hình mật khẩu. Tạo mật khẩu của bạn ngay tại đây — mình sẽ sinh hash để bạn dán vào file.

Hash của bạn
Cách áp dụng:
  1. Mở file admin/index.html trong repo.
  2. Tìm dòng: const ADMIN_PASS_HASH = '36bd3b2339c8a44467400600ae360c0195f3a5d36a38bdeeabbb633e80abcf03';
  3. Thay '__SET_ME__' bằng hash vừa sinh (giữ nguyên dấu nháy).
  4. Commit & push lên GitHub.
  5. Đợi GitHub Pages build xong (~1 phút), tải lại trang này.
Sau bước này, bất kỳ ai (kể cả bạn) vào /admin/ đều phải nhập đúng mật khẩu mới vào được.

Kết nối GitHub

Dán Personal Access Token để admin có thể lưu thay đổi (commit) vào kho. Token chỉ lưu trong trình duyệt này.

Tạo token an toàn (khuyến nghị):
  1. Mở Fine-grained tokens
  2. Resource owner = chủ repo · Repository access = Only select repositories → chọn đúng repo này
  3. Permissions → Repository → Contents: Read and write
  4. Expiration ngắn (vd 30–90 ngày), tạo token rồi dán vào ô trên
